The CSS Box Model

Every element is a box. Here's what's inside that box.

1. The Four Layers (Outside → Inside)

Every HTML element — a <div>, a <p>, an <img> — is a rectangular box with four layers. You can style each layer independently.

Key insight: Margin is outside the element — it pushes other elements away. Padding is inside the element — it pushes the content away from the edge. Border sits right between them.

2. Reading the Measurements

Here's the same diagram with explicit measurements. When you set padding: 25px, that's 25 pixels of space between the border and your content.

Common confusion: "Should I use margin or padding?"

Use padding when you want space inside an element (e.g., space between text and its background).
Use margin when you want space between elements (e.g., gap between two cards).

Quick Reference: Which Property Does What?

margin Space outside the element. Pushes other elements away. Can be negative. Can use auto for centering.
border The visible edge. Has width, style (solid, dashed, etc.), and color.
padding Space inside the element. Pushes content away from the edge. Takes on the element's background.
width/height By default, this sets the content size. With box-sizing: border-box, it includes padding and border.
box-sizing content-box (default): width = content only. border-box: width = content + padding + border. Use border-box.
