What Are Breakpoints?

Breakpoints are specific viewport widths where your layout changes. Bootstrap defines 6 breakpoints. When you use a class like col-md-6, the "md" means "at medium screens and up."

  • Below md, the column defaults to full width (12 columns)
  • To change behavior at smaller sizes, add more classes: col-12 col-md-6

üîë How Column Classes Work

  • col-12 = full width (12/12 columns) at all sizes
  • col-md-6 = half width (6/12) at medium and up, full width below
  • col-6 col-md-4 col-lg-3 = half on mobile, third on tablet, quarter on desktop
  • Columns should add up to 12 (or less) per row

üîë Display Utility Patterns

  • d-none = hidden at all sizes
  • d-block = visible at all sizes
  • d-none d-md-block = hidden on mobile, visible at md+
  • d-block d-lg-none = visible until lg, then hidden
  • d-none d-md-block d-xl-none = visible only between md and xl

min-width vs max-width

This is where people get confused. Here's the rule:

Type Syntax Meaning Use When
min-width @media (min-width: 768px) 768px and LARGER Mobile-first (Bootstrap's approach)
max-width @media (max-width: 767.98px) 767.98px and SMALLER Desktop-first or targeting small only

üîë Why 767.98px and not 767px?

  • Bootstrap uses .98 to avoid a 1px gap where both queries could match
  • min-width: 768px + max-width: 767.98px = no overlap
  • If you used 767px, a viewport exactly 768px wide would match BOTH queries!

Bootstrap's Exact Media Queries

CSS
/* Small devices (landscape phones, 576px and up) */
@media (min-width: 576px) { ... }

/* Medium devices (tablets, 768px and up) */
@media (min-width: 768px) { ... }

/* Large devices (desktops, 992px and up) */
@media (min-width: 992px) { ... }

/* X-Large devices (large desktops, 1200px and up) */
@media (min-width: 1200px) { ... }

/* XX-Large devices (larger desktops, 1400px and up) */
@media (min-width: 1400px) { ... }
CSS - Targeting Below a Breakpoint
/* Below sm (extra small only) */
@media (max-width: 575.98px) { ... }

/* Below md (sm and smaller) */
@media (max-width: 767.98px) { ... }

/* Below lg (md and smaller) */
@media (max-width: 991.98px) { ... }

/* Below xl (lg and smaller) */
@media (max-width: 1199.98px) { ... }

/* Below xxl (xl and smaller) */
@media (max-width: 1399.98px) { ... }
