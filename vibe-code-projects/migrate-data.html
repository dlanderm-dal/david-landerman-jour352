Migrate Learning Hub Data

Your current origin is:

Learning Hub data (comments, queued topics, custom instructions) is stored per-origin. Use this tool to move data between file:/// and http://localhost:3000.

Step 1: Export from current origin

Open this page in the origin that HAS your data. Click Export to copy all Learning Hub data.

Step 2: Import into new origin

Open this page in the origin you want to move data TO. Paste the exported data below and click Import.

Quick way (if both origins are accessible)

If you're switching from file:/// to localhost:3000:

  1. Open file:///path/to/migrate-data.html in a browser tab
  2. Click Export All Data — it copies to clipboard
  3. Open http://localhost:3000/migrate-data.html in another tab
  4. Click in the import box, press Cmd+V, click Import Data
  5. Done — all your comments, queue, and settings are now on localhost
