Learning Hub

Your interactive collection of study guides. Highlight text, ask questions, and build change files for AI-assisted learning.

How this works & how to set it up

The Learning Hub is designed for active engagement with content — not passive reading. After you complete the setup, you can import content, create a topic from scratch, then interrogate it. Highlight text and add comments just like Google Docs. Comments can be:

  • Questions about what you just read
  • Requests for clarification on a referenced topic
  • Changes to how an answer or topic is laid out
  • Anything! Write it as if you’re talking to an AI that can see the page

When ready, click Export Comments to copy a structured instruction to your clipboard, then paste it into Claude Code. It updates the website with your comments in mind.

Setup

Because this page is personalized and meant to be updated constantly — almost like an AI-powered word processor — it runs on a local server. Pick one option below:

Option A: One command

Open the Terminal app (Applications > Utilities > Terminal) and paste this. It does everything automatically, downloading to your Desktop.

git clone https://github.com/david-landerman/david-landerman-jour352.git ~/Desktop/learning-hub && cd ~/Desktop/learning-hub/vibe-code-projects && npm install && printf '#!/bin/bash\ncd "$(dirname "$0")"\necho "Starting Learning Hub..."\nopen "http://localhost:3000"\nnode image-server.js\n' > "Start Learning Hub.command" && chmod +x "Start Learning Hub.command" && echo "Done!"

Option B: Step by step

Open Terminal (Applications > Utilities) and run each command:

  1. git clone https://github.com/david-landerman/david-landerman-jour352.git ~/Desktop/learning-hub
    Downloads the repo to your Desktop.
  2. cd ~/Desktop/learning-hub/vibe-code-projects
    Enter the project folder.
  3. npm install
    npm is a package manager that downloads JavaScript tools & libraries. This installs what the Learning Hub needs. Everything it downloads is listed in package.json so you can audit it.

Make it easy to open:

  1. Paste this entire block into Terminal to create the launcher file:
    cat > ~/Desktop/learning-hub/vibe-code-projects/"Start Learning Hub.command" << 'EOF'
#!/bin/bash
cd "$(dirname "$0")"
echo "Starting Learning Hub..."
open "http://localhost:3000"
node image-server.js
EOF
chmod +x ~/Desktop/learning-hub/vibe-code-projects/"Start Learning Hub.command"
What if I don’t want it on my Desktop?

Move the learning-hub folder anywhere you like in Finder. The Start Learning Hub.command file uses relative paths, so it works from any location. Just double-click it wherever it ends up.

Security notes:

  • Server runs on localhost:3000 only — not exposed to the internet.
  • No data leaves your machine. No external services contacted.
  • The server accepts only validated image files and plain text notes.
  • All code is readable: image-server.js (~100 lines) and learning-hub.js.
